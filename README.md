\# Seoul Bike Usage Data Analysis



\## Project Overview

This project analyzes Seoul public bike rental data to understand usage patterns.



The analysis focuses on how time, weekday, and weather conditions affect bike rental demand.



\## Tools

\- Python

\- Pandas

\- Matplotlib



\## Dataset

The dataset contains bike rental information along with weather data.



Main variables used in this analysis:



\- datetime : date and time

\- temperature : temperature

\- humidity : humidity

\- count : bike rental count

\- weekday : day of the week



\## Analysis

The following analyses were performed:



1\. Monthly bike usage analysis

2\. Hourly bike usage analysis

3\. Bike usage by weekday

4\. Relationship between temperature and bike rentals

5\. Relationship between humidity and bike rentals

6\. Correlation analysis between variables



\## Key Findings

\- Bike usage increases during commuting hours.

\- Higher temperatures are associated with increased bike rentals.

\- Weekday usage tends to be higher than weekend usage.
